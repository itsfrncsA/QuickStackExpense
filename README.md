# QuickStack Expense Tracker

A full-stack expense tracking application built with the MERN Stack.

## Links

- **Live Demo:** https://quick-stack-expense-frontend.vercel.app
- **Frontend Repo:** https://github.com/itsfrncsA/QuickStackExpense-Frontend

## Features

- User Authentication (JWT)
- Expense Tracking with Categories
- Budget Management
- Savings Goals
- Philippine Peso (?) Support
- Responsive Design

## Tech Stack

- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Deployment: Vercel, Render
